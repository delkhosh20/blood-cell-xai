# Explainable Blood Cell Classification

## Project Overview

This project investigates deep learning and explainable AI techniques for blood cell classification using microscopy images.

The goals are:

- Compare multiple machine learning and deep learning approaches for blood cell classification.
- Evaluate models using accuracy, precision, recall, F1-score, and confusion matrices.
- Apply explainability techniques such as Grad-CAM to understand model predictions.
- Explore the tradeoff between predictive performance and interpretability in medical AI systems.

## Dataset

Blood Cells Image Dataset (Kaggle)

## Planned Models

- CNN Baseline
- ResNet50
- Additional transfer learning models (to be determined)

## Explainability
